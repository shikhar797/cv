Topic 1:Vector in C++

1.The C++ Standard Library vector class is a class template for sequence containers. A vector stores elements of a given type in a linear arrangement, and allows fast random access to any element.

2.To push a element in vector we use function push_back().

3.To delete a element in vector we use function pop().

Topic 2:Stack in C++

1.The C++ Standard Library stack class is a class template for sequence containers. A stack stores elements of a given type in a LIFO order.

2.To push a element in stack we use function push().

3.To delete a element in stack we use function pop().

Topic 3:Queue in C++

1.The C++ Standard Library queue class is a class template for sequence containers. A queue stores elements of a given type in a FIFO order.

2.To push a element in queue we use function push().

3.To delete a element in queue we use function pop().

Topic 4:Deque in C++

1.The C++ Standard Library dequeclass is a class template for sequence containers. A deque stores elements of a given type in both FIFO and LIFO order.

2.It works both as stack and queue .It have both property of stack and queue

3.To push a element in front we use function push_front().

4.To delete a element in front we use function pop_front().

5.To push a element in back we use function push_back().

6.To delete a element from back we use function pop_back().
